# AWS Highly Available Web Application

## Project Overview

Built a highly available web application infrastructure
using AWS CloudFormation.

## Architecture

<img src="/screenshots/Blank diagram.png" alt="diagram" width="900">

## Technologies

- CloudFormation
- VPC
- Subnets
- Route Tables
- Internet Gateway
- NAT Gateway
- Auto Scaling Group
- Application Load Balancer
- IAM
- EC2
- Nginx

## Features

- Multi-AZ deployment
- Public and private subnet segregation
- Auto Scaling
- Load Balancing
- Infrastructure as Code

## Deployment Steps

1. Deploy network stack
2. Deploy server stack
3. Verify target health
4. Access application through ALB
